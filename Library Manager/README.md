# CLI Book Manager

A simple command-line interface (CLI) application to manage a personal book collection.  
Allows adding, listing, removing, sorting, and updating books stored in a text file.

---

## Features

- Add or update books with **title, author, and pages read**.
- List all books in a readable format.
- Remove books by title.
- Sort books by **title, author, or pages read**, in ascending or descending order.
- Persistent storage using a local `books.txt` file.
- Lightweight CLI interface with menu-driven options.
